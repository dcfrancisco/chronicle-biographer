# Writing Studio

The Writing Studio is currently a prototype surface.

Current state
- It is not an API-backed writing workflow.
- It shows mock memory, observation, and chapter content.
- It is useful as a product sketch, not as shipped functionality.

Do not treat this as current MVP behavior
- Theme-to-chapter scaffolding
- Versioning and compare views
- Export presets
- Persona sliders
- Inline citation tooling

Roadmap usage
- Keep this doc as a future concept note until the writing workflow is API-backed and connected to the truthful MVP.
