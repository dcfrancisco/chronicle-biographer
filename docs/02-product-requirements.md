# Product Requirements - Chronicle

Version: 0.1
Date: 2026-06-18

Chronicle is an AI biographer. The current product is intentionally small: it turns uploaded artifacts into extracted text, memory candidates, observations, hypotheses, questions, and a chapter draft.

Current MVP
- Upload a single artifact.
- Extract text locally.
- Chunk content into analysis units.
- Create memory candidates from chunks.
- Generate observations.
- Generate hypotheses.
- Generate questions.
- Generate a chapter draft.

Product principles
- Evidence first: every claim should stay tied to source material.
- Simplify before expanding: keep the product honest about what works now.
- Preserve voice: the draft should not overwrite the user's intent.
- Push back with restraint: ask useful questions, but only after the core flow is solid.

Deferred features
- Interviews
- Timeline
- Relationships
- Themes dashboard
- Full notebook
- Book reader integration
- Workspace SaaS
- OIP provider
- Multi-provider switching
- OCR
- ASR
- Vector search
- Object storage
- Workers
- Kubernetes

Notes
- Anything outside the current MVP belongs in the roadmap or future-architecture docs, not in the main product narrative.
