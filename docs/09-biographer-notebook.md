# Biographer Notebook

The Biographer Notebook is currently a prototype surface.

Current state
- It is mock-driven.
- It presents observations, hypotheses, questions, contradictions, and themes as a concept layout.
- It does not represent a shipped notebook product.

Do not treat this as current MVP behavior
- Natural-language querying
- Notebook cell types
- Promotion workflows
- Snapshot export
- Shared notebook actions

Roadmap usage
- Keep this doc as a future concept note until the notebook is API-backed and clearly separated from mock data.
